# Social Network Simulation

A C++ simulation of a social network, built as a university project for the **Algorithms and Data Structures (ASP)** course.

## About

This project models a simplified social network where users can be connected to each other. The underlying structure uses a **graph** to represent relationships between users (nodes = users, edges = connections/friendships).

## Features

- Add and remove users
- Add and remove connections (friendships) between users
- Traverse the network (BFS / DFS)
- Find shortest path between two users
- Display the network / adjacency information
